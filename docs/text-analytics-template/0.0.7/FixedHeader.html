FixedHeader

FixedHeader class enables a fixed header appear on tables that have .reportal-fixed-header class when the table header is scrolled under address bar.

init()

Source:

Initializes the fixed header for a table by wrapping it into a div.aggregatedTableContainer, cloning the header and calculating its position and width. Besides it sets up listeners for a scroll event (to display the cloned header) and for a debounced resize event (to recalculate the header columns widths)
